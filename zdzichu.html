Połącz Google Calendar

Potrzebujesz darmowego klucza API od Google. Zajmie to około 10 minut i zrobisz to tylko raz.

Czego potrzebujesz:
· Konto Google
· Google Cloud Console (darmowe)
· ~10 minut

Krok 1: Utwórz projekt

1. Wejdź na console.cloud.google.com
2. Kliknij "Nowy projekt"
3. Nazwa: "Zdzichu"
4. Włącz Google Calendar API

Krok 2: Klucze

1. Dane uwierzytelniające → Klucz API
2. Identyfikator klienta OAuth → Aplikacja internetowa
3. Dodaj annalesiak.github.io jako autoryzowane źródło

Zdzichu na telefonie 📱

Strona działa pod adresem annalesiak.github.io/zdzichu/zdzichu.html — otwórz go na telefonie i dodaj do ekranu głównego.

iPhone: Safari → ikona udostępniania → "Dodaj do ekranu głównego"

Android: Chrome → trzy kropki → "Dodaj do ekranu głównego"
